goloops
=======

Here is some fun with loops in Go.

This is support for [the article at the blog](https://goingfullstack.com/backend/funny-thing-in-go-loops/).

Run the examples with:

```
$ go run with_error.go
```

and

```
$ go run with_hack.go
```

the output should be:

```
3
3
3
```

and

```
1
2
3
```
